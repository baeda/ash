## Dependencies
* [Java](https://www.java.com/) >= Version 8
* [Apache Maven](https://maven.apache.org/) >= 3.3.9
* [GCC](https://gcc.gnu.org/) >= 4.8

## Building
To build the project, run maven in the root directory:
```bash
mvn clean install
```
